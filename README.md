# a noname job posting portal , absoluty no name 

this is a job website, it allows employers to post their jobs for a fee, they can also opt to boost their job posting for a fee and allows the jobseekers to see the posted jobs, search for posted jobs and apply ( on the employer website ) for said posted jobs ...

## Employers:
    - can register.

    - can post a job.

        - with company logo and website 
        - with link to their application form

        - the job posting has a title, an excerpt , and full posting supporting markdown

        - boost the job ( this will push it up on the landing page of the site)

        - monitor engagement stats regarding the job posting ( clicks , views , applications )

        - posting the job happens for a fee 

        - the job posting is understood to be for 30 days 

        - the job boosting is understood to be for the same time period.

        - the job can be re posted as is or with modification for the same fee.
        
    - can cancle the posting , along with the boost if applicable before the end of the 30 days, there will be no refunds.
